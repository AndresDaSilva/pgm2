This program receives Error code locations in Hexidecimal values from a text file
Converts the hexidecimal value to binary
Converts the binary value to decimal
And identifies the CHIP number that contains the address of that error
Assuming we are using a 32gb RAM pc, we have 8 chips
We are not allowed automatic conversion methods or parse commands.
